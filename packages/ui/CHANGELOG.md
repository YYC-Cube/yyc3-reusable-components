## [1.0.0] - 2026-03-30

### Added

- Initial release of `@yyc3/ui` package
- Form components: `Button`, `Input`, `Checkbox`, `Switch`, `Select`, `Textarea`, `Slider`
- Layout components: `Card`, `Dialog`, `Sheet`, `Drawer`, `Sidebar`, `Tabs`
- Data display: `Table`, `Badge`, `Avatar`, `Progress`, `Skeleton`, `Tooltip`
- Navigation: `Breadcrumb`, `Pagination`, `NavigationMenu`, `Menubar`
- Overlay: `AlertDialog`, `Popover`, `HoverCard`, `ContextMenu`, `DropdownMenu`
- Advanced: `Calendar`, `Carousel`, `Chart`, `Command`, `GlassCard`

### Testing

- Unit tests for core components
- Storybook stories for documentation

### Accessibility

- Full ARIA support
- Keyboard navigation
- Screen reader compatible
