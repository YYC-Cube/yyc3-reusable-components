## [1.0.0] - 2026-03-30

### Added

- Initial release of `@yyc3/hooks` package
- AI hooks: `useAI`
- Navigation hooks: `useNavigationContext`
- Storage hooks: `usePersistedState`, `useChatPersistence`
- UI hooks: `useResponsive`, `useNotifications`, `useUISettings`
- DevOps hooks: `useDevOps`, `useChannelManager`, `useChannelConfig`
- Database hooks: `useDatabaseConfig`, `useSupabaseSync`
- Gesture hooks: `useGestures`

### Testing

- Unit tests for all hooks
- Storybook stories for interactive hooks
