## [1.0.0] - 2026-03-30

### Added

- Initial release of `@yyc3/ai` package
- AI chat components: `Chat`, `ChatContainer`, `ChatInput`, `ChatMessage`
- AI sidebar components: `ChatSidebar`, `ClaudeSidebar`
- AI panel components: `ArtifactsPanel`, `AsciiArt`
- Settings and system components: `SettingsModal`, `SystemStartup`
- Welcome and typing components: `ClaudeWelcome`, `TypingIndicator`
- Background component: `YYC3Background`

### Features

- Full TypeScript support with complete type definitions
- React 18+ support with hooks
- Customizable theming via CSS variables
- Responsive design support
