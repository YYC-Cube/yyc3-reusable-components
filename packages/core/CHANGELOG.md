## [1.0.0] - 2026-03-30

### Added

- Initial release of `@yyc3/core` package
- Internationalization (i18n) module with multi-language support
- Supported languages: Chinese (zh), English (en), Arabic (ar)
- Performance monitoring utilities
- Performance alert and benchmark modules

### Features

- Type-safe i18n with full TypeScript support
- Configurable locale detection and persistence
- Performance metrics collection and analysis
