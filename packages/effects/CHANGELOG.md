# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-03-30

### Added

- Cyberpunk effects components: `ParticleCanvas`, `GlitchText`, `NeonCard`, `CyberpunkWidget`
- Storybook stories for all cyberpunk effects
- Unit tests and integration tests for all components

### Changed

- Updated component exports to include new effects
- Improved TypeScript type definitions

## [1.0.0] - 2026-03-27

### Added

- Initial release of `@yyc3/effects` package
- 3D effects component: `_3DEffects`
- Micro interactions component: `MicroInteractions`
- Parallax scroll component: `ParallaxScroll`
