# YYC3-ASCII艺术字-单字版

## 彩色ASCII艺术字打印

支持6种ANSI颜色的艺术字输出：
- `\033[31m` - 红色
- `\033[32m` - 绿色
- `\033[33m` - 黄色
- `\033[34m` - 蓝色
- `\033[35m` - 紫色
- `\033[36m` - 青色
- `\033[0m` - 重置颜色

## 2. 文件读取功能

从外部文件读取并正确处理转义的ANSI颜色代码。

## 3. 预设示例集合

内置A、Y、C、3四个字符的彩色ASCII艺术字示例。

## A-Z字母设计规范

### 设计原则

1. **竖列格式**：每个字符使用6行文本表示
2. **彩虹渐变**：每行使用不同颜色（红→绿→黄→蓝→紫→青）
3. **等宽字符**：使用Unicode方块字符（█、╗、╝、╔、╚等）确保跨平台一致性
4. **居中对齐**：所有字符在同一垂直线上对齐

### 字符模板结构

每个字母遵循以下结构模式：

```python
'CHARACTER': [
    '\033[31m  第1行（红色）  \033[0m',
    '\033[32m  第2行（绿色）  \033[0m',
    '\033[33m  第3行（黄色）  \033[0m',
    '\033[34m  第4行（蓝色）  \033[0m',
    '\033[35m  第5行（紫色）  \033[0m',
    '\033[36m  第6行（青色）  \033[0m'
]
```
